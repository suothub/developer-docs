## Fluent

```php

use Suot\Fiscal\Cte\ValueObject\CteDeliveryFailureReason;

$now = (new DateTimeImmutable('now', new DateTimeZone('America/Sao_Paulo')))
    ->format('Y-m-d\TH:i:sP');

$payload = new CteDeliveryFailurePayload(
    authorizationProtocol: $nProt,
    attemptedAt: $now,
    reason: CteDeliveryFailureReason::ReceiverRefused,
    // reasonJustification: obrigatĂ³rio se reason === Other
    attemptNumber: '001',
    failedNfeKeys: [],
);

$result = $cte->deliveryFailure()
    ->forOperation('ie-1')
    ->inHomologation()
    ->accessKey($chave44)
    ->emitterCnpj($cnpj)
    ->deliveryFailure($payload)
    ->handle();
```

Motivos: `ReceiverNotFound`, `ReceiverRefused`, `AddressNotFound`, `Other`.

## Command equivalente

```php
use Suot\Fiscal\Cte\Application\Command\DeliveryFailure\RegisterCteDeliveryFailureCommand;

$command = RegisterCteDeliveryFailureCommand::create()
    ->forOperation('ie-1')
    ->inHomologation()
    ->accessKey($chave44)
    ->emitterCnpj($cnpj)
    ->deliveryFailure($payload)
    ->build();

$result = $cte->handle($command);
```
