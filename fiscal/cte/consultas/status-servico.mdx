## O quê / quando

Antes de emitir, confirme se o serviço de autorização da UF do emitente está em operação.

## Fluent

```php
use Suot\Fiscal\Core\ValueObject\Uf;

$result = $cte->serviceStatus()
    ->forOperation('status-1')
    ->inHomologation()
    ->issuerUf(new Uf('RS'))
    ->handle();

if ($result->isOperational()) {
    // seguir para emissão
} else {
    $failure = $result->failure();
}
```

Resultado: `CteStatusResult` (`serviceCode`, `message`, `isOperational()`, `failure()`).

## Query equivalente

```php
use Suot\Fiscal\Cte\Application\Query\ServiceStatus\GetCteServiceStatusQuery;

$query = GetCteServiceStatusQuery::create()
    ->forOperation('status-1')
    ->inHomologation()
    ->issuerUf(new Uf('RS'))
    ->build();

$result = $cte->handle($query);
```
