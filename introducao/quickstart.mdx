Este quickstart mostra o **modelo mental** do Fiscal SDK com CT-e. Você não precisa de token Composer para acompanhar. Quando for instalar no seu projeto, vá para [Instalação](/fiscal/instalacao).

<Info>
  **Pré-requisitos para só ler e entender:** nenhum.  
  **Para executar os scripts do monorepo:** clone `fiscal-sdk-php`, `composer install`, rode `php bin/dx/…` (stubs, sem SEFAZ).  
  **Para integrar no seu app:** PHP 8.4+, Composer 2, licença e token — [Instalação](/fiscal/instalacao).
</Info>

## O que você vai aprender

1. Dois entry points equivalentes: client `$cte` e `Command` + `Handler`
2. Rejeição fiscal é `Result`, não exception
3. O host persiste chave/protocolo/XML e decide retry

## Entry point A — Client fluent

```php
use Suot\Fiscal\Cte\Application\Cte;
use Suot\Fiscal\Core\ValueObject\Uf;
use Suot\Fiscal\Cte\Data\CteEmitter;
use Suot\Fiscal\Cte\Data\CteMunicipality;
// … demais Data objects

/** @var Cte $cte */
$result = $cte->issue()
    ->forOperation('ot-000123')
    ->inHomologation()
    ->numbered('1', '12345')
    ->issuedIn(new Uf('RS'))
    ->withOperation('5353', 'PRESTACAO DE SERVICO DE TRANSPORTE')
    ->dispatchedFrom($dispatchMunicipality)
    ->issuedBy($emitter)
    ->from($origin)
    ->to($destination)
    ->forShipper($shipper)
    ->deliverTo($receiver)
    ->carrying($cargo)
    ->charging($freight)
    ->taxedBy($tax)
    ->linkedToDocument($linkedNfe)
    ->underRntrc('12345678')
    ->handle();

if ($result->isAuthorized()) {
    // Host: gravar accessKey, protocolNumber, submittedDocumentXml
}

if ($result->isRejected()) {
    $failure = $result->failure();
    // Host: orientar usuário com $failure->recoveryGuide; retry só se canRetry()
}
```

Cadeia completa de dados de exemplo: `examples/cte/support/homologation-issue-command.php` no SDK. Passo a passo na docs: [Emissão de CT-e](/fiscal/cte/emissao).

## Entry point B — Command + Handler

```php
use Suot\Fiscal\Cte\Application\Command\Issue\IssueCteCommand;
use Suot\Fiscal\Cte\Application\Command\Issue\IssueCteHandler;

$command = IssueCteCommand::create()
    ->forOperation('ot-000123')
    ->inHomologation()
    // … mesma cadeia de domínio …
    ->build();

$result = $handler->handle($command);
```

Mesma cadeia tipada; só muda quem chama `handle()`.

## Pré-voo antes de emitir

```php
$validation = $cte->validate()
    ->issuerSetup()
    // … parâmetros do emitente …
    ->handle();

$status = $cte->serviceStatus()
    ->forOperation('preflight-1')
    ->inHomologation()
    ->issuerUf(new Uf('RS'))
    ->handle();

if (! $status->isOperational()) {
    // Não emita — consulte failure() / aguarde
}
```
